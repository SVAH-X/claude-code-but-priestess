内置普瑞赛斯 · Built-in backend

不依赖本地 CLI，直接连接任意 OpenAI 兼容服务（默认 LiteLLM；也支持 Ollama / LM Studio / vLLM / OpenRouter 等）。

可带或不带 /v1。LiteLLM 默认 http://127.0.0.1:4000；Ollama 为 http://127.0.0.1:11434/v1。
需与服务器上配置的模型名一致；点「测试连接」会自动列出可选模型。
隐私：以上配置只保存在本机数据目录的 settings.json 中，除了你填写的服务器之外不会发送给任何人。 API key is stored locally only and is sent only to the server you configure.
